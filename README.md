# Student-Marks-Tracker
A simple web application to manage students, subjects, and their marks. Built with ASP.NET MVC and EF core.

## Features:
- Add, edit, delete, and view all students
- Add and edit marks for multiple subjects
- View student averages
- Clean, responsive layout using Bootstrap
- Basic validation for scores (0-100)

## Database:
**Student Table**
- Id
- Name
- Email

**Mark Table**
- Id
- StudentId (FK)
- SubjectName
- Mark
